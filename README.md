# selkent-tracker
AFC Green Court Hub
Selkent U12 D-League Tracker 2025/2026 
Containing data on player stats and league/shield results

**iOS widget**
In the app: There's now a 📲 button in the Hub header (next to the 🌙 dark mode toggle). Tap it and it automatically reads all the live data — last result, league position, top scorer, next fixture — and copies a ready-to-paste DATA block to your clipboard.
In Scriptable: Open DukesWidget.js, find the const DATA = { block at the top, select from there down to the closing };, and paste. Done — the widget will refresh with the new data next time iOS updates it.
